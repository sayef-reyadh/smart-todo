🚀 HOW TO GET STARTED:

1. RUN A TUTORIAL:
   cd javascript-basics
   node 01-fundamentals.js
   node 02-variables.js
   # ... and so on

2. VIEW THE RESUME WEBSITE:
   Method A: Double-click basic-website/index.html
   
   Method B: Use local server
   - Windows: cd basic-website && python -m http.server 8000
   - macOS/Linux: cd basic-website && python3 -m http.server 8000
   - Then visit: http://localhost:8000

3. KEYBOARD SHORTCUTS (in website):
   Ctrl+A → About
   Ctrl+E → Experience
   Ctrl+S → Skills
   Ctrl+P → Projects
   Ctrl+C → Contact
