# Smart ToDo - User Personas

## Persona P1 - Aisha (University Student)
| Attribute | Details |
|---|---|
| Demographics | 21, undergraduate CS student |
| Goals | Submit assignments on time, balance study and part-time work |
| Motivations | Better grades, reduced stress |
| Pain Points | Overlapping deadlines, forgotten reminders |
| Technology Usage | Laptop-first, mobile web for quick checks |
| Typical Workflow | Capture tasks from LMS -> set due date/reminder -> filter by upcoming week |
